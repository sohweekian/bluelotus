12. HASHING AND CONSISTENCY
==============================================================================

Implement deterministic hash generation.

Recommended method:
1. Normalize capsule JSON with sorted keys.
2. Remove volatile fields if needed, or place volatile fields in a known section.
3. Compute SHA256.
4. Store as "capsule_hash".
5. Print same hash in JSON, TXT, Word, and Excel.

Pseudo-code:

import json, hashlib

def compute_capsule_hash(capsule: dict) -> str:
    stable = dict(capsule)
    stable.pop("capsule_hash", None)
    payload = json.dumps(stable, sort_keys=True, ensure_ascii=False, separators=(",", ":"))
    return hashlib.sha256(payload.encode("utf-8")).hexdigest()

Validation:
- TXT capsule hash must match JSON capsule hash.
- Word capsule hash must match JSON capsule hash.
- Excel capsule hash must match JSON capsule hash.
- If mismatch occurs, report generation should fail or mark INSTITUTIONAL_REVIEW_REQUIRED.
