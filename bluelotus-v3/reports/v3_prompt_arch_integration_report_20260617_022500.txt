  -> Quality Scorer
  -> Archive / prompt audit

PROMPT_ARCH_TEST_AGENT can restrict the new path to one agent for pilot runs.

6. Runtime Hardening Implemented
--------------------------------
  - Ollama client now returns failed raw model text on validation failure,
    allowing retry prompts to use the broken response.
  - Retry classifier now recognizes actual validator messages:
      not valid JSON
      missing required key
      must be number/string/boolean/array/object
      no more than N items
      enum errors
  - Prompt audit manifests are written for new-path agents:
      prompt_audit_<agent_id>.json
  - Raw compiled prompts are not saved by default.
  - New deterministic format normalizer applies only to new path outputs:
      missing key_finding evidence tags are prefixed
      missing risk priorities are prefixed
    This does not alter analytical content.
