NOTES FOR DR. CLAUDE CODE AND CHIEF ARCHITECT
-------------------------------------------------------------------------------
This upgrade changes BlueLotus V3 from a report-generation pipeline into a more
explicit deterministic institutional architecture.

The key improvement is not autonomy. It is memory, auditability, replayability,
and comparability.

V3.2 shows how the deterministic pipeline reasons from truth to advisory output.

The CIO remains the only execution authority.
