File:
  C:\bluelotus3\scripts\dashboard_widget_manager.py

Responsibilities:
  - Load and validate dashboard_widgets.yaml.
  - Render all enabled widgets from the registry.
  - Insert or replace only the protected thesis widget zone.
  - Verify S6/S7/S8 markers exist.
  - Verify widget order is stable.
  - Ensure the widget zone remains before System Health.
  - Fail publish when enabled widget markers are missing.
  - Preserve .nojekyll.

Key guard behavior:
  - If FAIL_PUBLISH_IF_ENABLED_WIDGET_MISSING is unset, default is fail-closed.
  - If enabled widget markers are missing, publisher/update scripts return
    failure instead of silently publishing a partial dashboard.
