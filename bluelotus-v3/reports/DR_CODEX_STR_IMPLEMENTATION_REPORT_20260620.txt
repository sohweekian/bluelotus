12. Known Limitations
---------------------
1. Source capacity remains CAPACITY_COLLECTING because resolved forecast history
   is still immature. This is intentional and thesis-compliant.

2. Cost-basis reconciliation cannot resolve ASTS/QUBT/LUNR/QBTS until a third
   independent witness is provided, such as broker statement export, second API
   endpoint, manual CSV, or approved screenshot/OCR extraction.

3. Kelly sizing is advisory only and depends on analyst upside and probability
   estimates. It is not a portfolio optimizer and not an order generator.

4. Hedge review uses available beta estimates and VXX/VIXY exposure. It should be
   treated as review intelligence, not a hedge instruction.

5. Dashboard publishing displays STR via Chief Strategist report text. A richer
   visual STR widget can be built later if CIO requests it.
